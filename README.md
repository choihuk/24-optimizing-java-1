## 진행방식

- 매주 수요일 스터디 전까지 모든 구성원이 한 챕터(혹은 두 챕터)씩 읽고 발표 자료의 형태로 정리하여 PR을 올립니다.
- 발표 당일 각 챕터별 발표자를 선정, 발표를 진행 후 질의응답 시간을 진행합니다.
- 발표자 선정시 이전 주차 발표자는 선정 과정에서 제외합니다.
